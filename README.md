# Malaria Forecasting (Bhutan)

A machine-learning project to forecast malaria incidence using WHO indicators.

## Streamlit App

This app:

✔ Loads & cleans WHO malaria indicators  
✔ Builds lag features  
✔ Trains a Linear Regression forecasting model  
✔ Predicts next year's malaria incidence
